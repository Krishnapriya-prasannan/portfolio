### Tailwind Configuration
Ensure dark mode is enabled in your `tailwind.config.js`:
```js
module.exports = {
  darkMode: 'class', // Enables dark mode
  content: ['./src/**/*.{js,jsx,ts,tsx}'],
  theme: {
    extend: {},
  },
  plugins: [],
};
```
